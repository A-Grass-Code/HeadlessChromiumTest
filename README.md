# HeadlessChromiumTest
Chromium 无头浏览器测试 Demo

说明：

1. 开发（完成）平台：**Win10 x64 / Win7 x64**；**.NET Framework 4.7** ；**VS 2017**。（ 其他环境未测试 ）
